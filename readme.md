## Instalation

 - [install npm](https://www.npmjs.com/get-npm)
 - [install docker](https://docs.docker.com/install/)

after

```code
    $ docker-compose build
    $ docker-compose up
```

or 

```
    $ npm run serve
```
and ready! the app is running on [`localhost:3000`](localhost:3000)
